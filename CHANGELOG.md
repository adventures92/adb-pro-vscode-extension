## [0.0.1] - 2023-10-27

### Added
- **Device Management Sidebar**: View connected devices, their status, and perform actions.
- **Smart Toggles**: Toggle Wi-Fi, Mobile Data, and Airplane Mode directly from the sidebar.
- **App Management**: Install APKs, uninstall apps, clear app data, and force stop apps.
- **Logcat Integration**: View and filter device logs by package and log level.
- **Screenshot Tool**: Capture device screenshots and save them to the local workspace.
- **Shell Access**: Run custom ADB shell commands.
- **Configuration**: `adb.path` setting to specify a custom ADB executable location.
